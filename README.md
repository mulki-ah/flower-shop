# Flower Shop Web Application 

## Project Description
This project is a simple flower shop web application built using HTML, CSS, and JavaScript.
It allows users to add flowers to cart and book flowers online.

## Features
- Add flowers to cart
- Remove items from cart
- Cart data stored using LocalStorage
- Flower booking form
- Input validation and user feedback

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

## How to Use
1. Open index.html in a browser
2. Add flowers to cart
3. Click cart icon to view items
4. Fill booking form and click Book Now

## Storage
- Cart and booking data are saved using browser LocalStorage
